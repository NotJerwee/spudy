# Spudy

Welcome to the repository of the project Spudy. Ongoing

# Installation

Clone the repository:
```
https://github.com/yourusername/spudy.git
```
or download zip folder.

Change to the project directory:
```
cd spudy
```

Install the necessary packages:
```
npm install
```

Start the project
```
npm start
```
